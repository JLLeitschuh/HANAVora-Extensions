Extensions
==========

Overview
--------

The extensions project includes SAP's SQL extensions to Apache Spark.

The new features include support for Hierarchical Queries, SAP currency conversion, extensions to
Apache Thrift Server, Apache Zeppelin, and Python bindings. Additionally, the package comes with
some startup scripts to facilitate starting extended spark processes.
